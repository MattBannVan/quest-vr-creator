# quest-vr-creator

Immersive VR Creation Experience for Meta Quest 3 - Built with A-Frame, hosted on GitHub Pages. **Everything done with Grok Using automations.**

## Key Features (Enhanced)
- Physics-enabled scene with super-hands controller interactions
- Intelligent object spawning: click tablet button to create cubes dynamically placed in front of the user
- Error mitigation hooks, app state management, try-catch throughout
- 3D tablet UI foundation ready for expansion
- Automated deployment via GitHub Actions

## Automation & Workflows
- **GitHub Workflow**: `.github/workflows/deploy-to-pages.yml` — On push to main, validates JS syntax (catches errors early), deploys to GitHub Pages. Ensures live site is always stable and error-reduced.
- **Grok Skill**: Custom `quest-vr-creator` skill in agent skills — provides structured workflows for analysis, implementation, rigorous testing (syntax + logic + local + VR), correction, and alignment to your values of intelligent, automated, precise VR development.

## Getting Started
1. Open in Meta Quest Browser (or desktop for dev)
2. Use controllers to interact with tablet button — spawns intelligent objects
3. Expand UI, add more tools via the skill-guided process

## Development
- Follow the quest-vr-creator skill's Local Development Workflow for all changes (analyze, plan, implement, test thoroughly, correct, deploy)
